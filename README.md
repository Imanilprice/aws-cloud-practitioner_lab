\## 📂 Project 1: My First Cloud Resume



\*\*Objective:\*\* Host a static HTML resume on Amazon S3 with public access.



\*\*Services Used:\*\*

\- Amazon S3 (Static Website Hosting)

\- AWS IAM (Bucket Policies)

\- AWS Cost Explorer (Monitoring)



\*\*Key Learnings:\*\*

\- How to create an S3 bucket with public access

\- How to enable static website hosting

\- How to write a bucket policy for public read access

\- Understanding the difference between Block Public Access and Bucket Policies
