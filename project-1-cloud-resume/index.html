Project 1: Static Resume Hosted on AWS S3

My first real AWS deployment. I built this page, configured an S3 bucket for static website hosting, wrote a bucket policy for public read access, and got a live URL on the internet. The goal wasn't just to follow a tutorial — it was to understand what each step actually does.

Amazon S3 Static Website Hosting IAM Bucket Policy HTML/CSS AWS Cost Explorer
What I actually learned
  • The difference between "Block Public Access" settings and a bucket policy — and why both matter
  • How S3 serves static files without a web server running anywhere
  • How to write a bucket policy JSON that grants s3:GetObject to anyone
  • How to monitor costs with AWS Cost Explorer (this site costs ~$0.00/month)
  • That keeping the bucket live (vs deleting it) preserves the link for a portfolio
